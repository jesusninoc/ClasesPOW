# Puntos a valorar de SEO
- El sitio web tiene que tener más páginas, no solo el index.html
- Tu web en más buscadores
- Comprobar que aparece en Google Analytics
- Crear robots.txt
- Las webs en la raíz del repositorio de Github
- Añadir sitemap.xml
- No tener páginas de errores, redirección si hay error
- Probar con más navegadores
- Enlaces de vuelta y de ida
- El sitemap tiene páginas que aparecen en Google
- Anuncios
- ¿El sitio es accesible?
- Redes sociales (LinkedIn)
- Dar de alta en Bing Webmasters Tools
- Meter un bot (chatbot)
- Crear banners
